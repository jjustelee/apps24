# apps24

Multilingual utility platform rebuilt for Vercel + Cloudflare DNS.

## Stack

- Next.js App Router
- TypeScript
- Vercel hosting
- Cloudflare DNS only

## Commands

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm run build
```

## Project rules

- Keep routes path-based.
- Add tools through the registry.
- Reuse the shared tool shell.
- Keep locale and SEO metadata in sync.
- Prefer small, surgical changes.
